# CS2 Case Opener Workspace

You can run commands from either location now:

- From workspace root (`c:/CS2CaseOpener`):
  - `npm run dev`
  - `npm run build`
- From app folder (`c:/CS2CaseOpener/cs2-case-opener`):
  - `npm run dev`
  - `npm run build`

## Important

Use Vite (`npm run dev`). Do not open `index.html` directly with `file://` in the browser.
